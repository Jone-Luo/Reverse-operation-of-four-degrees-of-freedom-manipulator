# Reverse-operation-of-four-degrees-of-freedom-manipulator

Our full video on YouTube can be found through this link: https://www.youtube.com/watch?v=idQu0Rw6mI0&t=79s

arm.py is the code for the reverse operation of the mechanical arm. It runs on Raspberry PI 4b.

All you need to do is look at the serial number of the bus steering gear and measure the length of each section of the mechanical arm and replace the parameter in the code.
